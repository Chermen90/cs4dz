# Горячие клавиши
1. ctrl + s - сохранить
2. ctrl + c - копировать
3. ctrl + v  - вставить
4. shift + win + s - сделать скриншот области
5. ctrl + x - вырезать
6. ctrl + l - очистка терминала
7. ctrl + z - отмена
    * ctrl + shift + z - отмена отмены 
8. win + d - буфер копирования
